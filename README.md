# Quantum Crypto

Why quantum computers can break elliptic curve cryptography but cannot break hash functions.

[![Live Demo](https://img.shields.io/badge/demo-quantum--crypto--three.vercel.app-cyan)](https://quantum-crypto-three.vercel.app)

## Overview

An animated single-page explainer covering the asymmetric impact of quantum computing on modern cryptography. The site walks through five key sections:

1. **Elliptic Curve Cryptography** -- how ECC works and why it is widely used
2. **Shor's Algorithm** -- how a quantum computer dismantles ECC's mathematical foundation
3. **Hash Functions** -- why one-way hashing resists quantum attack
4. **Grover's Algorithm** -- the limited speedup quantum offers against hashes
5. **Side-by-Side Comparison** -- animated bar charts contrasting quantum vulnerability

## Design & Tech

- **Stack:** Single HTML file, vanilla JavaScript, HTML5 Canvas animations
- **Aesthetic:** Dark cyberpunk theme with cyan / magenta / green color coding
- **Typography:** JetBrains Mono (code) + Instrument Serif (headings)
- **Animations:** Canvas-based particle and diagram animations throughout
- **Deployment:** Vercel
